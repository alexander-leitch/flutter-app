# Flutter App with Google Sign-In

A Flutter application for mobile and web that allows users to login via their Google Accounts.

## Built with Google Antigravity
This application was generated and built using **Google Antigravity**, an advanced AI coding assistant.

## Getting Started

### Prerequisites
- Flutter SDK
- Firebase Project with Google Sign-In enabled

### Configuration
Before running the app, you must configure Firebase:

1.  **Generate `firebase_options.dart`**:
    Run `flutterfire configure` in the project root.

2.  **Platform Specifics**:
    - **Android**: Place `google-services.json` in `android/app/`.
    - **iOS**: Place `GoogleService-Info.plist` in `ios/Runner/`.

### Running the App
```bash
flutter run
```
